# Calculatortutorial

MS Visual Studio C++ Tutorial in C for GCC/MingW. The code will work in VS2017 and later.

## Building

Use make to build.

```bash
make
```

## Cleaning

Use make to clean.

```bash
make clean
```

## Usage

Run the app in the terminal.

```bash
./calculator
```

```dos
calculator
```

Press Ctrl-C to exit.
