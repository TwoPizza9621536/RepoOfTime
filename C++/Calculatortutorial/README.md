# Calculatortutorial

MS Visual Studio C++ Tutorial for GCC/MingW. The code will work in VS2017 and later.

## Building

Use make to build.

```bash
make
```

## Cleaning

Use make to clean.

```bash
make
```

## Usage

Run the app in the terminal.

```bash
./calculator
```

```dos
calculator
```

Press Ctrl-C to exit.
