# Calculator

A calculator console app from C# tutorial for Visual Studio in C++ for GCC.

## Building

Type make to build the files.

```bash
make
```

## Cleaning

Use make same scripts to clean.

```bash
make clean
```

## Usage

Run the app in the terminal.

```bash
./calculator
```

```dos
calculator
```

Press Ctrl-C to exit
